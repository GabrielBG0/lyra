# Exporting

Lyra can export any song to plain text or PDF. Neither format requires Lyra to open, so exports are good for sharing, printing, or archiving.

## Export as plain text

Go to **File → Export as Plain Text**. Lyra prompts you to choose a save location, then writes a `.txt` file with all sections in order, each labeled with its type and name.

## Export as PDF

Go to **File → Export as PDF** (or press `Ctrl+P` / `⌘P`). Lyra prompts you to choose a save location and generates a formatted PDF.

## Including version history

Both export formats have an option to append your snapshot history to the end of the file. When enabled, each snapshot is listed with its date, note, and the full lyric content as it was at that point. This is useful for archiving the full creative process alongside the final lyrics.
