# Keyboard Shortcuts

On Mac, substitute `⌘` for `Ctrl` throughout.

## File

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| New Song | `Ctrl+N` | `⌘N` |
| Save | `Ctrl+S` | `⌘S` |
| Save Take (snapshot) | `Ctrl+Shift+S` | `⌘⇧S` |
| Close Song | `Ctrl+W` | `⌘W` |
| Export as PDF | `Ctrl+P` | `⌘P` |

## Edit

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Undo | `Ctrl+Z` | `⌘Z` |
| Redo | `Ctrl+Shift+Z` | `⌘⇧Z` |
| Copy section | `Ctrl+C` | `⌘C` |
| Cut section | `Ctrl+X` | `⌘X` |
| Paste section | `Ctrl+V` | `⌘V` |
| Find | `Ctrl+F` | `⌘F` |
| Find Next | `Ctrl+G` | `⌘G` |
| Find Previous | `Ctrl+Shift+G` | `⌘⇧G` |
| Toggle Replace | `Ctrl+Shift+H` | `⌘⇧H` |

## View

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Toggle Sidebar | `Ctrl+B` | `⌘B` |
| Toggle History Bar | `Ctrl+H` | `⌘H` |

## App

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Preferences | `Ctrl+,` | `⌘,` |
| Keyboard Shortcuts | `Ctrl+?` | `⌘?` |
