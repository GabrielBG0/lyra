# Lyra User Guide

Lyra is a desktop app for songwriters to write, version, and compare lyrics. Everything is stored in plain files on your machine, with no cloud sync or accounts required.

## Guides

- [Getting Started](getting-started.md) — Set up your vault and learn your way around the app.
- [Songs and Sections](songs-and-sections.md) — Create songs, add sections, and edit lyrics.
- [Snapshots](snapshots.md) — Save named versions of your work and browse your history.
- [Comparing Versions](comparing-versions.md) — Diff any two snapshots and pull changes between them.
- [Comments](comments.md) — Annotate sections with notes and track feedback.
- [Find and Replace](find-and-replace.md) — Search across your song and make batch replacements.
- [Exporting](exporting.md) — Export your lyrics as plain text or PDF.
- [Keyboard Shortcuts](keyboard-shortcuts.md) — Full reference.
